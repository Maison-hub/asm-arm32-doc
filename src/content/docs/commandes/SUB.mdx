## Syntax
<div class="cmd-syntax">
    <Code lang="asm" frame="terminal" code="SUB{cond} dest, src1, src2" />
</div>

L'instruction **SUB** est utilisée pour soustraire deux valeurs et stocker le résultat dans un registre.

```asm
SUB R0, R1, R2  ; Soustrait R2 de R1, stocke le résultat dans R0
``` 
## Example
```asm
SUB R1, R2, R3  ; Soustrait R3 de R2, stocke le résultat dans R1
```
