## Syntax
<div class="cmd-syntax">
    <Code lang="asm" frame="terminal" code="MOV{S}{cond} dest, op1 {, SHIFT_op #expression}" />
</div>

---


MOV qui signifie "move" (déplacer) est une instruction utilisée pour copier une valeur d'une source (op1) vers une destination (dest). L'instruction peut également inclure un décalage (SHIFT_op) appliqué à l'opérande source avant la copie.
```asm
MOV R0, R1          ; Copie la valeur de R1 dans R0
MOV R2, #10         ; Charge la valeur immédiate 10 dans R2
MOV R3, R4, LSL #2  ;Copie la valeur de R4 dans R3 après l'avoir décalée à gauche de 2 bits
``` 

## Example
```asm
MOV R2, #10; Charge la valeur immédiate 10 dans R2
```
